# 👑 جدول متابعة ملوك بايثون (بايثون من الصفر للاحتراف) - Medo KING

هذا الجدول هو خريطة الطريق الخاصة بي لمتابعة التقدم في لغة بايثون وتطويع المنطق البرمجي خطوة بخطوة.

---

## 1️⃣ الدوال الجاهزة الأساسية (Built-in Functions & Methods)
- [x] `print()` -> لطباعة النصوص والمخرجات على الشاشة.
- [x] `input()` -> لأخذ مدخلات من المستخدم.
- [x] `type()` -> لمعرفة نوع البيانات (هل هو رقم، نص، إلخ).
- [x] `len()` -> لحساب طول النص أو عدد العناصر جوه اللستة.
- [x] `int()` / `float()` / `str()` -> لتحويل البيانات من نوع لآخر (Casting).
- [x] `range()` -> لتوليد أرقام وراء بعضها (مهمة جداً مع الـ Loops).
- [x] `sum()` -> لجمع الأرقام بسرعة.
- [x] `max()` / `min()` -> لمعرفة أكبر وأصغر قيمة.
- [x] `.split()` -> لتقطيع النص الواحد لـ List من الكلمات أو الأرقام بناءً على المسافات.
- [x] `Slicing [start:end:step]` -> لتقطيع النصوص والـ Lists أو عكسها بـ سطر واحد سحري مثل `[::-1]`.
- [x] `.isupper()` / `.isdigit()` -> لفحص نوع الحروف والنصوص (كابيتال أو أرقام). *(added)*
- [ ] `round()` -> لتقريب الأرقام العشرية.
- [ ] `quit()` / `exit()` -> لإنهاء البرنامج فوراً وإلغاء العملية.

---

## 2️⃣ أساسيات التحكم والشرط (Control Flow)
- [x] `if` / `elif` / `else` -> الشروط والتحقق من البيانات.
- [x] الروابط المنطقية `and` / `or` / `not` -> دمج الشروط بذكاء.
- [x] `nested if` -> لوضع شرط داخل شرط آخر لتفريع القرارات.

---

## 3️⃣ الحلقات والتكرار (Loops)
- [x] `for loop` -> للتكرار عدد محدد من المرات (تلف على لستة أو رينج).
- [x] `while loop` -> للتكرار بناءً على شرط (تفضل شغال لحد ما الشرط يتغير).
- [x] `break` -> لإيقاف اللوب فوراً والخروج منها.
- [x] `continue` -> لتخطّي اللفة الحالية والدخول في اللفة اللي بعدها.

---

## 4️⃣ هياكل البيانات (Data Structures)
- [x] `Lists [ ]` -> لعمل قائمة من العناصر (مرتبة وقابلة للتعديل).
- [x] `append()` -> دالة مخصصة لإضافة عنصر جديد في آخر اللستة.
- [x] `insert()` -> لإضافة عنصر في مكان محدد (Index) جوه اللستة.
- [x] `remove()` / `pop()` / `clear()` -> دوال لمسح عناصر من اللستة أو تفريغها تماماً.
- [ ] `Dictionaries { }` -> لتخزين البيانات على شكل (مفتاح وقيمة Key:Value) كقاموس مرن.
- [ ] `keys()` / `values()` / `items()` -> دوال للتحكم في عناصر القاموس واستخراجها.

---

## 5️⃣ الدوال المخصصة (Custom Functions)
- [x] الكلمة السحرية `def` -> عشان تصنع دالة خاصة بيك أنت وتسميها بنفسك.
- [x] `Arguments & Parameters` -> تمرير البيانات المدخلة للدوال لتشغيلها ديناميكياً.
- [x] `return` -> لإرجاع قيمة من الدالة بعد ما تخلص شغلها لاستخدامها في مكان آخر.

---

## 6️⃣ معالجة الأخطاء (Error Handling)
- [x] `try` / `except` -> لحماية الكود من الـ Crash وإظهار رسائل تحذيرية محترمة.

---

## 7️⃣ البرمجة كائنية التوجه (OOP) 🎯
- [ ] `Classes & Objects` -> بناء هيكل الكائنات وتوليد نسخ منها.
- [ ] `Attributes & Methods` -> خصائص الكائنات والوظائف اللي بتعملها.
