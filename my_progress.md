# 👑 جدول متابعة ملوك بايثون (بايثون من الصفر للاحتراف)

## 1️⃣ الدوال الجاهزة الأساسية (Built-in Functions)
- [x] `print()` -> لطباعة النصوص والمخرجات على الشاشة (أنت عارفها أوريدي).
- [x] `input()` -> لأخذ مدخلات من المستخدم (أنت عارفها أوريدي).
- [x] `type()` -> لمعرفة نوع البيانات (هل هو رقم، نص، إلخ).
- [ ] `len()` -> لحساب طول النص أو عدد العناصر جوه اللستة.
- [x] `int()` / `float()` / `str()` -> لتحويل البيانات من نوع لآخر (Casting).
- [ ] `range()` -> لتوليد أرقام وراء بعضها (مهمة جداً مع الـ Loops).
- [ ] `sum()` -> لجمع الأرقام بسرعة.
- [ ] `max()` / `min()` -> لمعرفة أكبر وأصغر قيمة.
- [ ] `round()` -> لتقريب الأرقام العشرية.
- [ ] `quit()` / `exit()` -> لإنهاء البرنامج فوراً وإلغاء العملية.

---

## 2️⃣ أساسيات التحكم والشرط (Control Flow)
- [x] `if` / `elif` / `else` -> الشروط والتحقق من البيانات.
- [x] الروابط المنطقية `and` / `or` / `not` -> دمج الشروط (زي ما عملت في الأوبيريتور).

---

## 3️⃣ الحلقات والتكرار (Loops)
- [ ] `for loop` -> للتكرار عدد محدد من المرات (تلف على لستة أو رينج).
- [ ] `while loop` -> للتكرار بناءً على شرط (تفضل شغال لحد ما الشرط يتغير).
- [ ] `break` -> لإيقاف اللوب فوراً والخروج منها.
- [ ] `continue` -> لتخطّي اللفة الحالية والدخول في اللفة اللي بعدها.

---

## 4️⃣ هياكل البيانات (Data Structures)
- [ ] `Lists [ ]` -> لعمل قائمة من العناصر (تقدر تضيف وتمسح منها).
- [ ] `append()` -> دالة مخصصة لإضافة عنصر جديد جوه اللستة.
- [ ] `remove()` / `pop()` -> دوال لمسح عناصر من اللستة.
- [ ] `Dictionaries { }` -> لتخزين البيانات على شكل (مفتاح وقيمة) زي قاموس الكلمات.

---

## 5️⃣ الدوال المخصصة (Custom Functions)
- [ ] الكلمة السحرية `def` -> عشان تصنع دالة خاصة بيك أنت وتسميها بنفسك.
- [ ] `return` -> لإرجاع قيمة من الدالة بعد ما تخلص شغلها.

---

## 6️⃣ معالجة الأخطاء (Error Handling)
- [ ] `try` / `except` -> عشان لو الكود فيه غلطة البرنامج ميفصلش ويطلع رسالة محترمة للمستخدم.
